Бесплатная установка операционной системы УБУНТУ в Уручье



Убунту (Ubuntu) - это программная операционная система с открытым исходным кодом, которая работает от настольного компьютера до облака и всех ваших вещей, подключенных к Интернету.

Самая популярная в мире операционная система для общедоступных облаков и облаков OpenStack.

Самый быстрый путь от разработки до производства в IoT.

Платформа номер один для контейнеров от LXD до Kubernetes и Docker, Ubuntu может запускать ваши контейнеры в масштабе.

Быстрая, безопасная и простая Ubuntu поддерживает миллионы ПК по всему миру.

Подробнее - ubuntu.com

Что такое терминал (terminal) в Убунту?

По умолчанию терминал в Убунту (Ubuntu) и macOS запускает так называемую оболочку bash, которая поддерживает набор команд и утилит; и имеет свой собственный язык программирования для написания сценариев оболочки.

Для вызова терминала следует нажать сочетание клавиш - Ctrl-Alt-T.

Подробнее - ubuntu.com

Команды терминала. Часть 1.

Для вызова терминала следует нажать сочетание клавиш - Ctrl-Alt-T.

Введем в терминале первую команду - pwd. Комманды чувствительны к регистру, т.е. ввод - PWD выдаст ошибку.

Команда pwd скажет вам точно, какой текущий рабочий каталог. Вы можете изменить рабочий каталог с помощью команды cd ‘change directory’.

Попробуем ввести - cd /, pwd. Теперь ваш рабочий каталог - «/». Каталог «/», часто называемый корневым каталогом, является основой этой объединенной файловой системы. Оттуда все остальное разветвляется, образуя дерево каталогов и подкаталогов.

Из корневого каталога следующая команда переместит вас в «домашний» каталог (который является непосредственным подкаталогом «/»): cd home, pwd.

Чтобы перейти к родительскому каталогу, в данном случае обратно к «/», используйте специальный синтаксис из двух точек (..) при смене каталога (обратите внимание на пробел между cd и .., в отличие от DOS, вы не можете просто набрать CD .. как одна команда).

Команда whoami напомнит вам о вашем имени пользователя.

Для создания папки введите - mkdir /tmp/tutorial. В этом примере, в папке tmp появится папка - tutorial.

С помощью команды ls, можно узнать список файлов и папок в текущей деректории.

Подробнее - ubuntu.com

Команды терминала. Часть 2.

Для вызова терминала следует нажать сочетание клавиш - Ctrl-Alt-T.

Чтобы создать папку нужно ввести команду mkdir ‘make directory’. Например - mkdir /tmp/tutorial.

Перейдем в созданную папку - cd /tmp/tutorial. Далее в этой папке (tutorial) создадим три папки с помощью команды - mkdir dir1 dir2 dir3.

Подробнее - ubuntu.com

ГИМП (GIMP) - это бесплатный редактор изображений с открытым исходным кодом.

GIMP - это кроссплатформенный редактор изображений, доступный для GNU / Linux, OS X, Windows и других операционных систем. Это бесплатное программное обеспечение, вы можете изменить его исходный код и распространять свои изменения.

Являетесь ли вы графическим дизайнером, фотографом, иллюстратором или ученым, GIMP предоставляет вам сложные инструменты для выполнения вашей работы. Вы можете еще больше повысить свою производительность с помощью GIMP благодаря множеству параметров настройки и сторонних плагинов.
